## 项目介绍 ##
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;本项目是依赖于lufylegend库而制作的。结构比较简单、单一，利用lufylegend库件中的原生类构建游戏中的各个组件。如：Lsprite类构建背景层以及预览层，LLoader类加载图片等。
